= Outputs the contents of a bbPress forum into a clickable sitemap tree. =

== Description ==

Outputs the contents of a bbPress forum into a clickable sitemap tree:

Forum 1
 - Topic 1
    -Reply 1
    -Reply 2
    -Reply 3
    -Reply 4
 - Topic 2
    -Reply 1
    -Reply 2
    -Reply 3
    -Reply 4

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place '[bbpress_sitemap]' or '[bbpress_sitemap forum_id="0"]' in your post/pages
